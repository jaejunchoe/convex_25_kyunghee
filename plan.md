minimize: α·Σᵢ (cⱼ + Σᵢ xᵢⱼ / Cⱼ)² + β·Σᵢⱼ dᵢⱼ·xᵢⱼ + γ·Σᵢⱼ (pⱼ·tᵢ)·xᵢⱼ

subject to 
Σⱼ xᵢⱼ = 1, ∀I
Σᵢ xᵢⱼ ≤ Cⱼ - cⱼ, ∀j
xᵢⱼ = 0 if dᵢⱼ > Dₘₐₓ
xᵢⱼ ≥ 0, ∀i,j
(cⱼ + Σⱼ xᵢⱼ ) / Cⱼ ≤ ρ_max, ∀j


ᵢ: User
ⱼ: Parking number
xᵢⱼ: i를 j에 보낼 확률
dᵢⱼ: i가 j에 가는 거리
cⱼ: 현재 주차되어 있는 차량 수
Σᵢ xᵢⱼ: 새로 배정할 차량 수
Cⱼ: 주차장 j의 최대 수용 가능 수
pⱼ: 주차장 j의 시간당 비용
tᵢ: 이용자 i의 주차 시간
α: 주차장 효율성 (공간 활용도) 
β: 시민 만족도 (민원 최소화) 
γ: 사회적 비용 (시민 경제적 부담)
